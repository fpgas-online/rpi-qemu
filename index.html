QEMU RPi APT Repository

QEMU with Raspberry Pi 4B GENET Ethernet support.

Usage

# Add the repository
echo "deb https://fpgas-online.github.io/rpi-qemu trixie main" | \
  sudo tee /etc/apt/sources.list.d/qemu-rpi.list

# Import the signing key (if signed)
# curl -fsSL https://fpgas-online.github.io/rpi-qemu/repo-key.gpg | \
#   sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/qemu-rpi.gpg

# Install
sudo apt-get update
sudo apt-get install qemu-rpi-system-arm

Packages

  • qemu-rpi-system-arm - QEMU aarch64 emulator with RPi4B GENET
  • qemu-rpi-system-data - Data files for qemu-rpi

Source: github.com/fpgas-online/rpi-qemu
